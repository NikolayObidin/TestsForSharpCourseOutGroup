Приложение - мультиплеерная игра для смартфона в жанре файтинг.

	Автоматизируем:
					1. Регистрацию нового пользователя и авторизацию
					2. UI приложения
					3. Работа с внутриигровым магазином
					4. Проверка баланса новых юнитов
					5. Регрессионное тестирование для каждого обновления
	Не автоматизируем:
					1. Кроссплатформенность
					2. Локализация
					3. Установка и удаление приложения
					4. Удобство использования
					5. Новый функционал
